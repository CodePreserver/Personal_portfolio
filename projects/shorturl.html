Welcome to the URL Shortener Bot

Welcome to the **URL Shortener Bot**! Quickly shorten URLs using multiple services or use the default service for a fast result. Simply send a URL and I'll handle the rest, or use specific commands for different services.

Getting Started

To shorten a URL, follow these simple steps:

  1. Send a URL: Just send me a URL, and I'll shorten it using the default service.
  2. Use Specific Services: If you prefer a specific URL shortener, use the commands listed below.

Available Commands

The bot supports several commands to shorten URLs using different services. Use any of the following commands:

  • /tinyurl <URL>: Shortens the URL using TinyURL.
  • /isgd <URL>: Shortens the URL using is.gd.
  • /vgd <URL>: Shortens the URL using v.gd.
  • /clck <URL>: Shortens the URL using clckruURL.
  • /dagd <URL>: Shortens the URL using da.gd.
  • /cleanuri <URL>: Shortens the URL using CleanURI.

Unshorten URL

If you'd like to expand a shortened URL, simply use:

  • /unshorten <URL>: Expands a shortened URL back to its original form.

Custom API Service

If you'd like to use our custom API for direct URL shortening, just send the URL without any specific command, and I'll shorten it with our own service.

Example Usage

/tinyurl https://example.com - Returns a shortened URL using TinyURL.
